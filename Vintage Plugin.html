Vintage camcorder    plugin

Vintage Camcorder Plugin is an image processing filter plugin in .tox format for TOUCH DESIGNER  software, which can add vintage effect to your images and make your works more stylized.
Note: Please be advised that the Vintage Camcorder plugin requires configuration by the user to align with specific requirements. It does not include recording, post-production, or compression features.
